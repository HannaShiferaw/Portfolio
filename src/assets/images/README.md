# Assets / Images

- **logo.svg** – Used in header; favicon is served from `public/logo.svg`.
- **portrait** – Add your photo as `portrait.jpg` (or `.png`) in `public/` so the banner uses `/portrait.jpg`. The app falls back to a placeholder if the file is missing.
- **project-images/** – Add project screenshots here or in `public/project-images/` (e.g. `1.jpg`, `2.jpg`) and set `imageUrl` in `src/constants.js` to `/project-images/1.jpg`, etc.
